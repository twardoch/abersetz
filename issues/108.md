## Task 108.1.3

- Analyze ./external/htmladapt/ 
- Incorporate usage of htmladapt into abersetz (HTML translation mode). Chunking should work appropriately and smartly. The idea is that pushing full HTML into the LLM is not a good idea, so htmladapt "slims" it by externalizing out certain things into a separate file, and then being able to merge it back into the original HTML, or into the translated HTML. So we ask the models to only translate the slimmed HTML. You have write access to the ./external/htmladapt/ folder so you may also edit it to match our needs (the package is new, not yet used, so no need to worry about breaking existing code). The package ultimately goes to https://github.com/twardoch/htmladapt and to https://pypi.org/project/htmladapt/
